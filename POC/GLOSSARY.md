# GLOSSARY — [AI]mbassy

Сокращения и термины проекта. Кросс-язычный: законы Австрии на немецком, ответы на русском.
Пополняем по ходу. Источник правды по нормам — сами законы в корпусе (`data/laws/`).

## Организации
- **НП — «Народные Посольства»** (People's Embassies). Беларусская диаспоральная НКО, для которой делается проект; ведёт реальные юридические консультации. Поставляет корпус и эталонную разметку eval. Фигурирует в эталонных ответах эксперта.
- **BBU** — Bundesagentur für Betreuungs- und Unterstützungsleistungen. Австрийское госагентство правовой помощи/опеки (юристы по убежищу). Названо в кейсах эксперта; в ответах бота не хардкодим (дисклеймер универсальный).
- **AMS** — Arbeitsmarktservice. Служба занятости Австрии (выдаёт разрешения на работу иностранцам — см. AuslBG).
- **RIS** — Rechtsinformationssystem des Bundes. Официальная БД законов Австрии (`ris.bka.gv.at`); источник всех URL законов.

## Законы корпуса (law_code → что это)
- **NAG** — Niederlassungs- und Aufenthaltsgesetz. Закон о поселении и пребывании (ВНЖ).
- **NAG-DV** — NAG-Durchführungsverordnung. Подзаконный акт к NAG (какие документы прикладывать к заявлению; переходные положения).
- **StbG** — Staatsbürgerschaftsgesetz. Закон о гражданстве.
- **FPG** — Fremdenpolizeigesetz. Закон о полиции по делам иностранцев (паспорта иностранца, депортация, refoulement).
- **AsylG** — Asylgesetz 2005. Закон об убежище (статус беженца, субсидиарная защита).
- **AuslBG** — Ausländerbeschäftigungsgesetz. Закон о занятости иностранцев (разрешения на работу).

## Юридические термины
- **§ / Abs. / Z** — параграф / Absatz (абзац) / Ziffer (пункт). Адрес нормы: «§ 11a Abs. 4 Z 2».
- **BGBl.** — Bundesgesetzblatt. Официальный вестник законов (нумерация редакций).
- **Aufenthaltstitel** — вид на жительство (общий термин).
- **RWR / RWR+** — Rot-Weiß-Rot-Karte (plus). «Красно-бело-красная» карта — ВНЖ с доступом к рынку труда.
- **Daueraufenthalt** — статус долгосрочного резидента ЕС.
- **Fremdenpass** — паспорт иностранца (выдаётся при отсутствии действующего паспорта, § 88 FPG).
- **non-refoulement / Refoulementverbot** — запрет высылки в страну, где грозит опасность (§ 50 / § 45a FPG, ст. 2/3 EMRK).
- **EMRK** — Europäische Menschenrechtskonvention (Европейская конвенция по правам человека). НЕ в корпусе (наднациональный акт).

## Техника (проект)
- **SICRT** — метод оценки retrieval из курса (дек 8): golden segment list + recall-калькулятор. У нас = разметка волонтёров + Hit Rate@k.
- **self-preference bias** — судья-LLM завышает ответы своей же модели-генератора. Лечим панелью из 3 судей (Gemini+Claude+Codex).
- **gold_chunks / gold_chunks_secondary** — в `eval/testset.jsonl`: точные §-эталоны / «тоже засчитывается» (глава-уровень, напр. 8. Hauptstück FPG для q07).
